# Sistema de Citas Médicas

Este proyecto es un **sistema web de gestión de citas médicas**, desarrollado con **Laravel**  y **Angular** . Permite gestionar pacientes, doctores, horarios y citas médicas de manera eficiente y centralizada.

---

## Tecnologías utilizadas

- **Backend:** Laravel 
- **Frontend:** Angular 
- **Base de datos:** MySQL  

---

## Funcionalidades principales

- Registro y gestión de pacientes  
- Gestión de doctores y especialidades  
- Agenda de citas médicas  
- Visualización de estadísticas y dashboards
